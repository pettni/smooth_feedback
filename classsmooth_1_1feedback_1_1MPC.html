Detailed Description

template<std::size_t K, typename T, LieGroup G, Manifold U, typename Dyn, smooth::diff::Type DiffType = smooth::diff::Type::DEFAULT>
class smooth::feedback::MPC< K, T, G, U, Dyn, DiffType >

Model-Predictive Control (MPC) on Lie groups.

Template Parameters
Knumber of MPC discretization steps (seee ocp_to_qp).
Ttime type, must be a std::chrono::duration-like
Gstate space LieGroup type
Uinput space Manifold type
Dyncallable type that represents dynamics
DiffTypedifferentiation method

This MPC class keeps and repeatedly solves an internal OptimalControlProblem that is updated to track a time-dependent trajectory defined through set_xudes().

Note
If the MPC problem is nonlinear a good (re-)linearization policy is required for good performance. See MPCParams for various options. If none of the options are enabled it is up to the user to update the linearizaiton in an appropriate way, or make sure that the problem is linear. The linearization point can be manually updated via the linearization() member functions.

If MPCParams::relinearization_interval is set to a value \( I > 0 \), an internal counter is maintained that relinearizes the problem at every \( I \) calls to this function.

If MPCParams::iterative_relinearization is set to a value \( I > 0 \), this function checks up to \( I \) times whether the solution touches the linearization domain, and relinearizes and re-solves if that is the case.
