Detailed Description

template<LieGroup G, Manifold U>
struct smooth::feedback::ASIFProblem< G, U >

Active set invariance problem definition.

The active set invariance problem is

\[ \begin{cases} \min_u & (u \ominus u_{des})' W_u (u \ominus u_{des}) \\ \text{s.t.} & x(0) = x_0 \\ & h(x(t)) \geq 0, \quad t \in [0, T] \\ \end{cases} \]

for a system \( \mathrm{d}^r x_t = f(x(t), u(t)) \).
