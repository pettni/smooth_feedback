Block sparse matrix construction.

Parameters
blockslist of lists {{b00, b01}, {b10, b11 ...}} of (optional) sparse matrix blocks
Returns
the blocks as a single sparse matrix

Non-present (std::nullopt-valued) blocks are considered zeros.

Warning
The block sizes must be consistent, i.e. all blocks in the same block-column must have the same number of columns, and similarly for row-columns and rows.
