Convert an active set invariance problem to a QuadraticProgram.

The objective is to impose constraints on the current input \( u \) of a system \( \mathrm{d}^r x_t = f(x, u) \) s.t.

\[ \frac{\mathrm{d}}{\mathrm{d}t} h(\phi(t; x_0, bu(\cdot))) \geq \alpha h(\phi(t; x_0, bu(\cdot))) \]

which enforces forward invariance of the set \( \{ x : h(x) \geq 0 \} \) along the backup trajectory \( bu \). The constraint is enforced at \(K\) look-ahead time steps \( t_k = k \tau\) for \(k = 0, \ldots, K \).

This function encodes the problem as a QuadraticProgram that solves

\[ \begin{cases} \min_{\mu} & \left\| u - u_{des} \right\|^2 \\ \text{s.t.} & \text{constraint above holds for } u = u_{lin} + \mu \end{cases} \]

A solution \( \mu^* \) to the QuadraticProgram corresponds to an input \( u_{lin} \oplus \mu^* \) applied to the system.
