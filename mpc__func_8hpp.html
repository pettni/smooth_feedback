Active set invariance problem definition.

Allocate QP sparsity pattern (part 1 of ocp_to_qp()).

Allocate QP matrices (part 1 of asif_to_qp())

The active set invariance problem is

\[ \begin{cases} \min_u & (u \ominus u_{des})' W_u (u \ominus u_{des}) \\ \text{s.t.} & x(0) = x_0 \\ & u \in ulim \\ & h(x(t)) \geq 0, \quad t \in [0, T] \\ \end{cases} \]

for a system \( \mathrm{d}^r x_t = f(x(t), u(t)) \).

Parameters
[in]Knumber of constraint instances
[in]nu_ineqnumber in inequalities in input constraint
[in]nhnumber of barrier constraints
[out]qpallocated QP with zero matrices

Variables: [x_1, ..., x_K, u_0, ..., u_{K-1}]

Constraints:

  • Dynamics constraints (K * Nx)
  • Input constraints (Nu_ineq * Nu)
  • State constraints (Nx_ineq * Nx)
  • State linearization constraints (K * Nu) [optional]
