Result of Mesh function.

A generic function is a mapping (t0, tf, q, X, U) -> R^M

The function variables are NAME SIZE t0 1 tf 1 X nx*(N+1) U nu*N

Let the total number of variables be K = 2 + nq + nx*(N+1) + nu*N.
