Detailed Description

template<Manifold M>
struct smooth::feedback::ManifoldBounds< M >

Manifold constraint set.

The set consists of all \( m \) s.t.

\[ m \in M = \{ l \leq A ( m \ominus c ) \leq u \}. \]
